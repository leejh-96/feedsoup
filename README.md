# "feedsoup"
### 프로젝트 소개
자신의 이력서나 포트폴리오, 더 나아가 자유롭게 Communication 하는 Board 기반의 Web-Site
### 개발인원
총 1명(백엔드)
### 개발기능
로그인, 이메일 인증, 회원가입, 공지사항, 자유게시판, 문의게시판
